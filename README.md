# ServiceDesk

ServiceDesk - веб-приложение для обработки пользовательских заявок.

## Технологии

- Python
- Flask
- Flask-Login
- Flask-WTF
- Flask-SQLAlchemy
- SQLite
- Bootstrap

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

После запуска откройте страницу:

```text
http://127.0.0.1:8080
```

База данных создается автоматически в папке `data`.
Загруженные файлы сохраняются в папке `static/uploads`.

## Основные функции beta v1

- регистрация пользователя;
- вход и выход из аккаунта;
- личный кабинет;
- создание заявки;
- список заявок текущего пользователя;
- подробная страница заявки;
- хранение данных в SQLite;
- защита заявок от просмотра другими пользователями.

## Основные функции beta v2

- админ-панель `/admin`;
- роль администратора через поле `is_admin`;
- первый зарегистрированный пользователь автоматически становится администратором;
- обычный пользователь видит только свои заявки;
- администратор видит все заявки;
- администратор может менять статус заявки;
- администратор может удалять заявку;
- к заявке можно прикрепить файл;
- разрешенные расширения файлов: `png`, `jpg`, `jpeg`, `pdf`, `txt`;
- прикрепленный файл отображается на странице заявки.

## Как проверить админку

1. Запустите проект командой `python app.py`.
2. Откройте `http://127.0.0.1:8080`.
3. Зарегистрируйте первого пользователя.
4. Войдите под этим пользователем.
5. Откройте `/admin`.
6. Создайте несколько заявок.
7. В админ-панели проверьте, что видны все заявки.
8. Измените статус заявки.
9. Удалите тестовую заявку.

Если база уже была создана раньше, при запуске приложение добавит новые поля
и сделает первого пользователя администратором, если администраторов еще нет.

## Как проверить загрузку файлов

1. Войдите в аккаунт.
2. Откройте страницу создания заявки.
3. Заполните форму и выберите файл с расширением `png`, `jpg`, `jpeg`, `pdf`
   или `txt`.
4. Создайте заявку.
5. Откройте страницу заявки и проверьте ссылку на прикрепленный файл.
6. Проверьте, что файл появился в папке `static/uploads`.
7. Попробуйте загрузить файл с другим расширением и проверьте, что форма
   показывает ошибку.
