# ServiceDesk

ServiceDesk - веб-приложение для обработки пользовательских заявок.

## Технологии

- Python
- Flask
- Flask-Login
- Flask-WTF
- Flask-SQLAlchemy
- SQLite
- Bootstrap

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

После запуска откройте страницу:

```text
http://127.0.0.1:8080
```

База данных создается автоматически в папке `data`.

## Основные функции beta v1

- регистрация пользователя;
- вход и выход из аккаунта;
- личный кабинет;
- создание заявки;
- список заявок текущего пользователя;
- подробная страница заявки;
- хранение данных в SQLite;
- защита заявок от просмотра другими пользователями.

## Тестовый сценарий

1. Запустите проект командой `python app.py`.
2. Откройте `http://127.0.0.1:8080`.
3. Зарегистрируйте первого пользователя.
4. Войдите под этим пользователем.
5. Создайте заявку.
6. Откройте список заявок и подробную страницу заявки.
7. Выйдите из аккаунта.
8. Зарегистрируйте второго пользователя.
9. Проверьте, что второй пользователь не видит заявки первого.
